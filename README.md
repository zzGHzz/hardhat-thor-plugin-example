# hardhat-thor-plugin-example

Example of using [`hardhat-thor-plugin`](https://github.com/zzGHzz/hardhat-thor-plugin).

## Usage

To run the created task:

```bash
npx hardhat accounts
```

To run the test:

```bash
npx hardhat test
```

To run the script:

```bash
npx hardhat run script/deploy.ts
```
